# Roblox-Sensitivity-Analysis
UCSB Econ 134a Final Project 
This project is an interactive sensitivity analysis website for Roblox (ticker: RBLX). 
The values are defaulted to my DCF on Excel.
Here you can customize the WACC, Terminal Growth, and FCF Growth, along with the net debt and total shares outstanding.
